=== Секция {project_applicant}

Особенностью секции {project_applicant} является автоматическое заполнение всех обязательных полей данными из Единого государственного реестра юридических лиц (ЕГРЮЛ) или Единого государственного реестра индивидуальных предпринимателей (ЕГРИП), в зависимости от выбранного профиля. 
ifdef::manual3[В этой секции допускается заполнение и редактирование только необязательных полей.]
Следует учитывать, что время наполнения полей данной секции сведениями ЕГРЮЛ/ЕГРИП может занять некоторое время.

ifdef::manual15[В этой секции необходимо дозаполнить данные, обязательные для заполнения поля отмечены звёздочкой (*).]

Если заявку подает Физическое лицо, данные заполняются из учетной записи Госуслуг.

Если у юридического лица имеются филиалы, то при переходе в секцию «Заявитель» пользователю будет показано модальное окно, в котором можно выбрать один из филиалов организации, как на <<id_image-g_ca_10, Рисунке {counter:figure-number}>>. 

[[id_image-g_ca_10]]
.Рисунок {counter:figure-number}. Модальное окно «Выбор филиала»
image::g_ca_10.png[{image-size1}, align="center"]

Необходимо выбрать один из филиалов или использовать основные данные организации.
При выборе филиала, следующие поля будут автоматически заполнены данными, соответствующими выбранному филиалу:

* Полное наименование юридического лица;
* Сокращенное наименование юридического лица;
* КПП;
* Местонахождение на территории РФ.

ifdef::manual15[]
В случае, когда необходимо ввести дополнительные данные, для которых нет полей в форме, дополнительные поля следует добавить в форму самостоятельно. 
Этот функционал доступен, если рядом с полем находится кнопка для добавления, как на <<id_image-g_caa_8, Рисунке {counter:figure-number}>>.

[[id_image-g_caa_8]]
.Рисунок {counter:figure-number}. Кнопка добавления нового поля
image::g_caa_8.png[{image-size1}, align="center"]
 
Для добавления поля необходимо заполнить первое поле и только после этого нажать на активную кнопку «Добавить» – после этого в форме ниже добавится пустое поле или блок для заполнения, как на <<id_image-g_caa_9, Рисунке {counter:figure-number}>>.

[[id_image-g_caa_9]]
.Рисунок {counter:figure-number}. Активная кнопка для добавления
image::g_caa_9.png[{image-size1}, align="center"]

endif::[]

ifdef::manual3[]
Для всех типов профилей обязательной является активация чек-боксов в блоке «Требования к заявителю», как на <<id_image-g_ca_29, Рисунке {counter:figure-number}>>. 

[[id_image-g_ca_29]]
.Рисунок {counter:figure-number}. Подтверждение соответствия требованиям
image::g_ca_29.png[{image-size1}, align="center"] 

Активируя их, участник заверяет о соблюдении и соответствии данным требованиям. 
В противном случае участие в отборе на получение субсидии будет невозможно.
Все остальные незаполненные поля не являются обязательными, однако их заполнение рекомендуется.
endif::[]
