=== {profile_section}

==== {profile_info}

В разделе {profile_info} требуется указать основные данные. 
Информация о субсидии, сформированная в данном разделе, будет опубликована наряду с информацией, полученной из Реестра субсидий Подсистемы.

{icon_hint_1} (1). 
{icon_hint_2} (2), как на <<id_image-g_se_1_1, Рисунке {counter:figure-number}>>. 

[[id_image-g_se_1_1]]
.Рисунок {counter:figure-number}. Иконка подсказки и текст подсказки
image::g_se_1_1.png[{image-size1}, align="center"]

При редактировании раздела {profile_info} необходимо заполнить/редактировать следующие поля:

* {name_sub_1}.
* «Предельный размер субсидии для одного получателя» – выбор соответствующего значения из справочника:
** «установлено» — необходимо ввести сумму в соседнем окне, тогда во всех созданных отборах по этой субсидии будет использоваться введенный предельный размер субсидии (будет настроен автоматический контроль не превышения предельного размера заявителем при формировании заявки);
** «не установлен» — предельный размер во всех созданных отборах по этой субсидии не устанавливается;
** «в размере произведенных затрат» — предельный размер во всех созданных отборах по этой субсидии будет всегда равен произведенным затратам, который указывает {role_us} при формировании заявки;
** «устанавливается в объявлении об отборе» — предельный размер устанавливается свой в каждом отдельном отборе по этой субсидии.
* «Необходимо привлечение софинансирования, %» – выбор соответствующего значения из справочника. 
При выборе значения «диапазон» необходимо в дополнительных полях указать числа «от» и «до». 
При выборе любого из значений «не более»/«не менее»/«равно» заполняется одно дополнительное поле. 
При выборе значения «не требуется» дополнительных полей для заполнения нет.
* «Описание субсидии» – заполняется вручную.
* «География» – заполняется вручную или выбором значения из справочника «ОКТМО», можно выбрать несколько значений. По умолчанию, если поле оставить пустым, то значение будет автоматически проставлено «Российская Федерация».
* «Теги (ключевые слова, характеризующие субсидию)» – заполняется выбором значения из справочника.
* Кнопка «+ Медиаматериал» (1) имеет функционал добавления материалов и описания к ним.

[[media_materials]]
Созданный блок автоматически получает название «Медиаматериалы N», где «N» – это порядковый номер созданного блока. 
Каждое последующее нажатие на кнопку создает новый блок с полями. 
Любой из созданных блоков можно перемещать верх-вниз по списку блоков медиаматериалов и удалять.

Для использования этого функционала необходимо навести курсор мыши на блок, справа сверху отобразятся иконки, как на <<id_image-g_se_1_3, Рисунке {counter:figure-number}>>:

* image:{iconsdir}up.png[width=16,height=16] – если блок нижний в списке;
* image:{iconsdir}down.png[width=16,height=16] – если блок верхний в списке;
* image:{iconsdir}up.png[width=16,height=16] и image:{iconsdir}down.png[width=16,height=16] – если блок промежуточный в списке;
* image:{iconsdir}delete.png[width=16,height=16] – отображается у всех блоков.

{save_auto_check}.

В разделе {profile_requirements} можно отредактировать список требований, которые будут предъявляться к участникам отборов в рамках данной субсидии. 
{role_val} в последующем проводит проверку и контроль поданных заявок на соответствие установленным требованиям.

====
[%unbreakable, grid=none, frame=none, cols="1,15"]
|===
^.^| image:{iconsdir}info.png[16,16] |
Если в отборе на получение данной субсидии предполагается оценка заявок по показателям и критериям оценки с присвоением балла, то формирование перечня показателей и критериев осуществляется непосредственно при формировании объявления об отборе, а не в профиле субсидии.
|===
====

{requirements_decree}.

Список требований возможно дополнить двумя способами, как на <<id_image-g_se_1_8, Рисунке {counter:figure-number}>>: 

. Создание нового требования, не входящего в справочник – кнопка «+ Требование» (1).
. Выбор значения из справочника требований – кнопка «Выбрать из справочника» (2) (справочник содержит требования, которые установлены постановлением № 1781, но они не являются обязательными и необходимость их применения к субсидии устанавливается правилами предоставления субсидии, утвержденными нормативными правовыми актами).

Для добавления требований не из справочника необходимо нажать кнопку «+ Требование». 
Справа появится окно с полями для заполнения, как на <<id_image-g_se_1_9, Рисунке {counter:figure-number}>>.
 
[[id_image-g_se_1_9]]

В окне необходимо заполнить обязательные поля:

* {text_for_user};

{but_create}.
После нажатия кнопки «Создать» новое требование появляется в конце списка требований.

Для добавления требований из справочника необходимо нажать кнопку «Выбрать из справочника». 
В поле выбора нажатием левой кнопки мыши добавьте необходимые требования в список, как на <<id_image-g_se_1_10, Рисунке {counter:figure-number}>>.

Требования в порядке их добавления будут появляться в конце списка требований.
После того, как требования были добавлены, необходимо нажать кнопку «Завершить выбор» справа от поля выбора. 

Всем требованиям присваивается один из трех признаков, который отображается иконкой в начале текста требования, как на <<id_image-g_se_1_11, Рисунке {counter:figure-number}>>:

* Иконка image:{iconsdir}pin.png[width=16,height=16] – нельзя удалить и редактировать (1);
* Иконка image:{iconsdir}document.png[width=16,height=16] – ранее было выбрано из справочника, можно отменить выбор, нельзя редактировать (2);
* Иконка image:{iconsdir}document_add.png[width=16,height=16] – ранее было создано на Портале, можно удалить, можно редактировать (3).
