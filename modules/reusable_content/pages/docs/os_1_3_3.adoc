В блоке 
ifdef::role2[«Срок проведения экспертизы»]
ifdef::role2_cz[«Срок оценки заявок»]
необходимо настроить даты начала и окончания, в поле «Количество дней на экспертизу» цифра изменяется динамически. 

В поле «Провайдер подписания» необходимо выбрать из справочника:

* «Простая электронная подпись (ЕСИА)» – эксперт может попасть на Портал только авторизовавшись через ЕСИА, завершает он экспертизу нажатием кнопки в интерфейсе; 
* «Усиленная квалифицированная электронная подпись» – эксперт может попасть на Портал только авторизовавшись через ЕСИА и при завершении экспертизы должен подписать свое экспертное заключение УКЭП.

«{role_exp}» — это общая системная роль, под которой могут работать в том числе пользователи, которые являются членами комиссии.
