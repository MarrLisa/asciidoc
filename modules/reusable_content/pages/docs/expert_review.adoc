В экспертизе все критерии группируются в рамках видов:

* Качественные;
* Стоимостные. 

Каждый вид критерия состоит из критериев или критериев с показателям. 
Для раскрытия вида критерия необходимо нажать на иконку image:{iconsdir}down.png[width=16,height=16] (1), как на <<id_image-g_er_cz_7, Рисунке {counter:figure-number}>>. 
Если вид критерия содержит критерии с показателями, то отображается список критериев, содержащий название критерия (2), динамически изменяемое количество баллов по критерию (3) и иконки image:{iconsdir}down.png[width=16,height=16] (4) для раскрытия списка показателей (5). 
Каждый показатель содержит текст показателя с иконкой подсказки (6), вид оценки (7) и поле для комментария по данному показателю (8).

image::g_er_cz_7.png[{image-size1}, align="center"]

Если вид критерия содержит просто критерии, то при раскрытие по иконке image:{iconsdir}down.png[width=16,height=16] (1) будет отображаться список критериев (2), каждый из которых содержит текст критерия с иконкой подсказки (3), вид оценки (4) и поле для комментария по данному критерию (5), как на <<id_image-g_er_cz_8, Рисунке {counter:figure-number}>>.

Критерии или показатели содержат один из четырех видов оценки, как на <<id_image-g_er_cz_9, Рисунке {counter:figure-number}>>:

* прогрессивную шкалу с ползунком (1): позволяет оценить значение в заданном диапазоне, для этого необходимо перемещать ползунок внутри этого диапазона для выбора конкретного значения, или выставить значение в правом окне;
* дискретную с баллами (2): предоставляет возможность выбора из ограниченного набора значений, для этого необходимо выбрать одно из предложенных значений;
* список (3): представляет собой поле выбора с перечнем значений в выпадающем списке необходимо выбрать одно из предложенных.
* шкала типа «Да/нет» (4) с весовым значением: служит для расчёта балла по критерию, для этого выставляется вес критерию, а также правая и левая границы.

[[id_image-g_er_cz_10]]
.Рисунок {counter:figure-number}. Выставление оценки в показателе и динамический подсчет баллов критерия, вида критерия и общего

Пример понятно иллюстрирует процесс, где установлена пока только одна оценка 20 (1). 
Эксперт видит шкалу оценок без учета веса для каждого показателя.

Для критерия с показателями балл вычисляется как сумма баллов по каждому показателю внутри группы, умноженная на вес группы _[Формула: ((Оценка по показателю 1 * Вес показателя 1) + (Оценка по показателю N * Вес показателя N)+ (Оценка по показателю N+1 * Вес показателя N+1)) * Вес критерия с показателем]_ поэтому балл по критерию с показателями – 2,50 (2).

Для вида критерия балл вычисляется как сумма баллов по каждому критерию внутри вида _[Формула: (Балл по критерию с показателем 1) + (Балл по критерию с показателем N) + (Балл по критерию с показателем N+1)]_ поэтому балл по виду критерия – 2,50 (3).

Общий балл оценки заявки вычисляется как сумма баллов по всем видам критериев _[Формула: (Балл по виду 1 + Балл по виду N + Балл по виду N+1)]_, так как в примере выставлена только одна оценка, то общий балл 2,50 (4).

Если вид критерия содержит просто критерии, то при выставлении оценки в критериях (1) и (2) динамически изменяется балл в поле слева у вида критерия (3) и общий балл по блоку «Оценки заявки» (4).
