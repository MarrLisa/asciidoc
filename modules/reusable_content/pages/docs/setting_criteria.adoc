// Описание настройки критериев оценки заявки
// используется в os_1_3_3

В блоке «Критерии» добавляются и настраиваются критерии, по которым в последующем эксперты будут проводить оценку заявок. 
Для добавления критерия необходимо нажать кнопку «+ Критерий», как на <<id_image-g_os_1_3_3_3, Рисунке {counter:figure-number}>>. 
 
[[id_image-g_os_1_3_3_3]]
.Рисунок {counter:figure-number}. Кнопка «+Критерии»
image::g_os_1_3_3_3.png[{image-size1}, align="center"]
 
После нажатия открывается боковое окно «Создание критерия», где необходимо в поле «Шаблон» выбрать один из двух шаблонов критерия, как на <<id_image-g_os_1_3_3_4, Рисунке {counter:figure-number}>>:

* {crit_std} – предназначен для критериев, оцениваемых без учета конкретных показателей;
* {crit_std_w_ind} – применяется в случае, когда необходимо учитывать дополнительные показатели при оценке критерия.

При наведении курсора мыши на максимальный балл критерия с показателями будет отображаться подсказка с весовым значением критерия, как на <<id_image-g_os_1_3_3_17, Рисунке {counter:figure-number}>>.

[[id_image-g_os_1_3_3_17]]

При выставлении оценок экспертами система автоматически будет рассчитывать балл по критерию с учетом весовых значений как показателей, так и критерия. 
У критерия с показателями каждый показатель имеет свое собственное весовое значение, и сам критерий также обладает своим весовым значением. 
При расчете балла по критерию для каждого показателя система умножает выбранную оценку по показателю на весовое значение, заданное в показателе. 
Затем суммирует эти баллы всех показателей и умножает получившуюся сумму на весовое значение критерия. 

====
[%unbreakable, grid=none, frame=none, cols="1,15"]
|===
^.^| image:{iconsdir}warning.png[16,16] |
Таким образом, формула расчета балла по критерию с показателями выглядит следующим образом:
Балл по критерию = (Оценка по показателю 1×Весовое значение показателя 1) + (Оценка по показателю 2× Весовое значение показателя 2) + …+ (Оценка по показателю n × Весовое значение показателя n) × Весовое значение критерия.
Где: +
– «Оценка по показателю n» – это оценка по конкретному показателю, выставленная экспертом. +
– «Весовое значение показателя n» – это весовой коэффициент для соответствующего показателя. +
– «Весовое значение критерия» – это весовой коэффициент для всего критерия.
|===
====

* Вес со значением «Контрольный» (1), считается как сумма весов всех критериев и критериев с показателями; +
* Балл «Контрольный» (2), считается как сумма максимальных баллов всех критериев (3).

===== Настройка баллов

После того, как будут внесены все критерии необходимо настроить проходной балл по отбору. 
Для этого необходимо в блоке нажать иконку image:{iconsdir}edit.png[width=16,height=16] в строке «Балл» показатель «Проходной», как на <<id_image-g_os_1_3_3_19, Рисунке {counter:figure-number}>>.

[[id_image-g_os_1_3_3_19]]
.Рисунок {counter:figure-number}. Иконка редактирования баллов
image::g_os_1_3_3_19.png[{image-size2}, align="center"]
 
После нажатия открывается боковое окно «Настройка проходного балла» в котором необходимо заполнить поле «Проходной балл», значение при достижение, которого, проект будет считаться победителем. 
Значение не может быть более 100 (более значения «Максимальный балл»). 
После этого нажать кнопку «Применить», как на <<id_image-g_os_1_3_3_20, Рисунке {counter:figure-number}>>.

[[id_image-g_os_1_3_3_20]]
.Рисунок {counter:figure-number}. Настройка проходного балла
image::g_os_1_3_3_20.png[{image-size2}, align="center"]
 
Есть несколько условий, которые важно соблюсти для верного заполнения блока «Критерии»:

* Общий вес показателей в критерии по завершению добавления всех показателей в критерий должен равняться 1.
* Вес со значением «Контрольный» по завершению добавления всех критериев должен равняться 1.
* Значения «Максимальный» и «Контрольный» должны совпадать.

Чтобы проверить правильность заполнения можно воспользоваться кнопкой в верхней части страницы «Проверить форму» (1), после нажатия в форме подсветятся все допущенные ошибки (2,3,4), как на <<id_image-g_os_1_3_3_21, Рисунке {counter:figure-number}>>.
