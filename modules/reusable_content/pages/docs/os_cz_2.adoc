=== {otbor_section}

{otbor_section} содержит подразделы с необходимой информацией для проведения отбора:

. {profile_otb_info} – вносится информация по отбору, дополняется данными к той, что указана в услуге.
. {fund_services} – вносится информация по распределению средств между получателями социального заказа согласно внесенным данным. 
. {add_info} – при необходимости вносится информация, которая не отображена в основных разделах.

[[profile_otb_info]]
==== Подраздел {profile_otb_info}

В подразделе {profile_otb_info} формируется информация, которая будет отображаться в объявлении об отборе на публичной части Портала. 
Часть данных уже предзаполнены данными из услуги, эти данные невозможно редактировать в рамках отбора. 
На странице есть следующие поля и блоки:

* «Наименование» – полное наименование УО, подгружается из услуги, не редактируется.
* «Почтовый адрес» – почтовый адрес УО, подгружается из услуги, редактируется.
* «Адрес электронной почты» – электронный почтовый адрес УО, необходимо заполнить.
* блок «Контактные телефоны» состоит из полей:
** «Контактная информация» – заполняется вручную.
** «Телефон» – заполняется вручную, через маску, возможен только ввод цифр.
** Кнопка «+ Телефон» – при нажатии появляется новый блок «Контактные телефоны». 
Все добавленные блоки можно менять местами и удалять с помощью функциональных кнопок–иконок, которые появляются если навести курсор мыши на блок, как на <<id_image-g_os_cz_2_2, Рисунке {counter:figure-number}>>.

В подразделе {fund_services} производится детальная настройка услуг в социальном заказе. 
На странице размещен список социальных заказов (1), выбранных при создании отбора. 
Под каждым номером социального заказа расположен раскрывающийся список услуг (2), относящихся к социальному заказу и выбранных при создании отбора, как на <<id_image-g_os_cz_2_12, Рисунке {counter:figure-number}>>.

[[id_image-g_os_cz_2_12]]
.Рисунок {counter:figure-number}. Страница {fund_services}
image::g_os_cz_2_12.png[{image-size1}, align="center"]
 
Для настройки услуг необходимо воспользоваться иконкой image:{iconsdir}down.png[width=16,height=16] рядом с каждой услугой. 
При нажатии на эту иконку услуга раскрывается, и отображаются заголовки и атрибуты этой услуги. 
В этом режиме можно внести необходимые изменения и настройки для каждой отдельной услуги.

Для указания направления услуги и связанных с ним кодов ОКВЭД необходимо в строке «Направление и ОКВЭД» нажать кнопку «+ Направление». 
После этого появится поле «Направление», в котором будет отображаться только то направление (–я) ОКВЭД, которое соответствует (–ют) выбранному направлению услуги.

После выбора направления отобразится множественное поле выбора «ОКВЭД», в котором будут отображаться только те коды ОКВЭД, которые соответствуют выбранному направлению. 
Если у услуги имеется несколько направлений, то для добавления еще одного направления можно также воспользоваться кнопкой «+ Направление», как на <<id_image-g_os_cz_2_13, Рисунке {counter:figure-number}>>.

[[id_image-g_os_cz_2_13]]
.Рисунок {counter:figure-number}. Кнопка «+ Направления», когда к услуге есть несколько направлений
image::g_os_cz_2_13.png[{image-size1}, align="center"]

Однако, если все направления уже были выбраны для данной услуги, кнопка «+ Направление» не будет отображаться.

Указанные коды ОКВЭД будут влиять на выборку отборов, которые система автоматически сделает для участника отбора на публичной части портала в случае, если он захочет воспользоваться фильтрами и автоподбором. 

Ниже в блоке услуги присутствует таблица объемов, которая состоит из колонок, как на <<id_image-g_os_cz_2_14, Рисунке {counter:figure-number}>>:

* «Показатель объема» – это значение берется из версии услуги и отображает определенный показатель, связанный с объемом предоставляемой услуги.
* «Плановый объем» – это значение берется из версии услуги и показывает запланированный объем предоставляемых услуг.
* «Остаток объема услуги» — это значение, изначально берется из версии услуги. 
После заполнения поля в следующей колонке «Объем в отборе», сумма динамически пересчитывается по формуле: _остаток = плановый объем – объем в отборе_. 
Если результат является отрицательным, цифры подсвечиваются красным цветом, и рядом с числом отображается красный треугольник.
* «Объем в отборе» – это числовое поле, в котором необходимо вручную ввести количество объема, которое надо выбрать для данной услуги в текущем отборе.
* «Минимальный объем» – это числовое поле, в котором необходимо вручную указать минимально допустимое количество объема для данной услуги в текущем отборе для участника отбора.
* «Предельный объем» – это числовое поле, в котором необходимо вручную указать максимально допустимое количество объема для данной услуги в текущем отборе для участника отбора. 
Введенное значение не должно превышать остаток объема услуги и не может быть меньше значения, указанного в поле «Минимальный объем».

Любой из созданных блоков можно перемещать верх–вниз по списку блоков «Документ» и удалять, функциональные кнопки появляются при наведении курсора мыши на блок.
