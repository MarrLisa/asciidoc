При активных отборах, при создании заявки пользователю для заполнения откроется только активный шаблон с учётом его типа субъекта экономической деятельности.

Если шаблон уже настроен, его необходимо активировать, переведя переключатель из положения «Шаблон не активен» в активное положение.
Появится диалоговое окно «Активация шаблона» с предупреждением «После активации конкурсной процедуры внесение изменений будет невозможно. 
Убедитесь, что вы полностью заполнили шаблон заявки». 
Для активации шаблона {but_confirm}.

Если шаблон еще не настроен, данный раздел остается без изменений.

Под переключателем есть кнопка для выведения печатной формы данного шаблона в формате PDF «Распечатать шаблон».

===== Раздел «Дополнительные настройки» настройки шаблона заявки

Для перехода в раздел «Дополнительные настройки» необходимо на странице «Настройки шаблона» нажать на пункт в боковом меню.

После перехода в раздел «Дополнительные настройки» открывается страница настроек, как на <<id_image-g_os_1_3_2_1_2, Рисунке {counter:figure-number}>>:

. Кнопка «Сохранить» – для сохранения всех внесенных изменений (1);
. Блок «Текстовые формы» (2) – в котором можно изменить «Заголовок печатной формы заявки», при нажатии кнопки «^» (3) открывается поле ввода текста с возможностью его форматировать (4).

|===
^.^| image:{iconsdir}warning.png[16,16] |
Заголовок будет отображаться в печатной версии заявки, которую будет заполнять пользователь.
|===
====

В блоке «Текстовые формы», в поле «Заголовок печатной формы» прописана универсальная фраза «Заявка № {0} на участие в отборе 
ifdef::role2[на предоставление субсидии»]
ifdef::role2_cz[по социальному заказу»]
, где номер будет заполняться динамически номером заявки, которую захотят распечатать. 
При необходимости текст можно изменить, но не рекомендуется удалять «шорт код» [Заявка № {0}].

Если вносились изменения на данной странице необходимо их сохранить нажатием кнопки «Сохранить», в ином случае пропустить этот шаг.
