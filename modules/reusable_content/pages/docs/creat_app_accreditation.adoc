== Создание заявки на получение аккредитации

Аккредитация заполняется только в тех случаях, если организация/ИП/физическое лицо собирается принять участие в отборах социального заказа. Для участия в отборах по субсидиям аккредитация не требуется.

Для пользователей, которые ранее не подавали заявку на получение аккредитации, существует возможность подать заявку, как через публичную часть портала, так и через личный кабинет пользователя в разделе {my_applications}.

=== Создание заявки с публичной части Портала

==== Неавторизованный пользователь

Если ранее пользователь не был авторизован на Портале, как описано в xref:start_us.adoc#auth_portal[xrefstyle=full] настоящего руководства пользователя, то после перехода на страницу {navigator} через верхнее меню (1), будет отображаться баннер аккредитации с заголовком «Проверьте аккредитацию на портале» (2), как на <<id_image-g_caa_1, Рисунке {counter:figure-number}>>.

[[id_image-g_caa_1]]
.Рисунок {counter:figure-number}. {navigator} для неавторизованного пользователя
image::g_caa_1.png[{image-size}, align="center"]
 
При нажатии на баннер, в этой же вкладке браузера откроется страница авторизации. Необходимо пройти шаги, описанные в xref:start_us.adoc#auth_portal[xrefstyle=full] настоящего руководства.

==== Авторизованный пользователь

После успешной авторизации, как указано в xref:start_us.adoc#auth_portal[xrefstyle=full] настоящего руководства, при переходе на страницу {navigator}, будет отображаться баннер с актуальным статусом аккредитации.

Возможны следующие баннеры со статусами по типу пользователя (ФЛ/ИП/ЮЛ) в реестре аккредитованных участников:

* {ban_valid} – отображается, если аккредитация на портале действительна. 
В баннере есть возможность скрыть/раскрыть информацию, при раскрытии – отображается дата окончания аккредитации и кнопки «Изменить данные» для обновления информации и «Аннулировать» для исключения из реестра.
* {ban_expires} – отображается, если аккредитация на Портале заканчивается в течение следующих шести месяцев. 
В баннере есть возможность скрыть/раскрыть информацию, при раскрытии – отображается дата окончания аккредитации и кнопки: «Продлить» для продления аккредитации, «Изменить данные» для обновления информации и «Аннулировать» для исключения из реестра.
* {ban_revoked} – отображается, если аккредитация на Портале аннулирована, запрещает участие в отборах на оказание государственных (муниципальных) услуг в социальной сфере. 
В баннере есть возможность запросить аккредитацию через кнопку «Получить».
* {ban_noaccr} – отображается, если аккредитация на Портале не найдена, запрещает участие в отборах на государственных (муниципальных) услуг в социальной сфере. 
В баннере есть возможность запросить аккредитацию через кнопку «Получить», как на <<id_image-g_caa_2, Рисунке {counter:figure-number}>>.

== Заполнение заявки на получение аккредитации 

Страница заявки на получение аккредитации состоит из следующих элементов, как на <<id_image-g_caa_7, Рисунке {counter:figure-number}>>:

* информационная строка, содержащая уникальный номер заявки и дату создания (1), статус заявки (2) и кнопку вывода формы заявки в печать (3);
* {page_application2} (4);
* {page_application3} (5);
* {page_application4} (6).

[[id_image-g_caa_7]]
.Рисунок {counter:figure-number}. Страница заявки на получение аккредитации 
image::g_caa_7.png[{image-size}, align="center"]

include::autocheck_pic.adoc[]

Форма заявки имеет три секции:

* {project_applicant} – основные данные о заявителе;
* {participant_requirements} – заверение о соблюдении и соответствии данным требованиям;
* {project_apply_accr} – секция для подачи заявки после ее заполнения.
