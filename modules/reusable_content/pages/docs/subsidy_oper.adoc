.Рисунок {counter:figure-number}. Страница реестра «Субсидии (операторы)»

В таблице со списком профилей субсидий отображаются дополнительно две колонки и кнопка:

. «Родительская субсидия» – субсидия, которую получил Оператор для последующего предоставления иным лицам, с возможностью перехода на публичную страницу профиля;
. Кнопка «К субсидии» – по нажатию происходит переход в реестр {subsidy} с примененным фильтром в колонке «Наименование субсидии» (кнопка отображается только для ГРБС и не доступна для ГРБС (оператор)).
. «Оператор» – юридическое лицо, которое получило субсидию для последующего предоставления иным лицам.
