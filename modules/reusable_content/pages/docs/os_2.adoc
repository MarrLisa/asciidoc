// Раздел 2. Объявление в настройке отбора
// используется в 2 рп
{otbor_section} содержит подразделы с необходимой информацией для проведения отбора:

. {profile_otb_info} – вносится информация по отбору, дополняется данными к той, что указана в субсидии.
. {fund_distr} – вносится информация по распределению средств между получателями субсидии согласно внесенным данным. 
. {add_info} – при необходимости вносится информация, которая не отображена в основных разделах.

В этом подразделе необходимо указать сумму бюджетных средств, которую планируется распределить по победителям формируемого отбора (распределить средства по годам и КБК), эти данные будут отображены в реестре «Управление субсидиями» в колонке определенного года(–ов) в колонке «Принимаемые бюджетные обязательства». 
В полях «Год начала» и «Год окончания» выбираются значения из справочника «Бюджетный цикл» на какой год или года из трехгодичного цикла будут распределяться средства. 
Доступны для выбора только те года, которые указаны в субсидии, по которой организуется отбор. 
Данные года будут впоследствии отражаться в протоколах. 
В случае невозможности предоставления субсидии в пределах указанного в заявке периода финансирования в связи с недостаточностью лимитов бюджетных обязательств субсидия может предоставляться в очередном финансовом году без повторного прохождения отбора, если такое предусмотрено правилами. 
В таком случае, для распределения финансирования такой заявке за пределами указанного в ней периода, необходимо активировать чек–бокс «Возможность распределения средств за пределами указанного в заявке периода финансирования», как на <<id_image-g_os_2_13, Рисунке {counter:figure-number}>>.

В образовавшейся таблице необходимо заполнить поля сумм по годам, которые ранее были указаны в графе «Период предоставления мер поддержки», как на <<id_image-g_os_2_16, Рисунке {counter:figure-number}>>.

В поле «Остаток к распределению» выводится информация о доступном остатке средств на выбранном КБК в разрезе выбранных годов. 
Чтобы избежать ввода суммы превышающий остаток по КБК при вводе значений в поля автоматически выполняется пересчет суммы остатка на выбранном КБК. 
Если пересчитанная сумма остатка становится отрицательной, она выделяется красным цветом. 
Кроме того, рядом с отрицательной суммой появляется значок «треугольник с восклицательным знаком», по которому при наведении мыши выводится подсказка «Превышен лимит остатка к распределению».

Если хотя бы в одном поле «Остаток к распределению» из выбранных годов будет отрицательное значение, система запретит сохранение и выделит данные с отрицательными значениями красным цветом. 
Также, при отправке на согласование отбора, система проверяет остаток средств на выбранном КБК. 
Если остаток стал отрицательным, отправка отбора на согласование запрещается, и выводится модальное окно с соответствующим сообщением.

Далее представлен блок «Результаты предоставления субсидии», таблица в этом блоке заполняется автоматически сразу после добавления КБК в блоке выше. 
Таблица содержит информацию о результатах в разрезе нескольких лет. 
Значения для выпадающего списка берутся из Результатов версии субсидии, «привязанных» к выбранным КБК, в рамках которой создается отбор.

Рядом с каждым годом в таблице в блоке «Наименование результата» и в блоке «Показатели» есть чек–боксы, по умолчанию они все активированы. 
Если деактивировать чек–бокс, то этот год не будет отображаться на публичной странице Объявления об отборе в секции {profile_otb_info} в блоке «Результаты предоставления субсидии».

^.^| image:{iconsdir}info.png[16,16] |
Выбор чек–бокса является обязательным для результатов и для показателей. 
То есть, если деактивировать все чек–боксы, либо активировать их только в одном из блоков, то при нажатии кнопки «Проверить обязательные поля», чек–боксы будут подсвечены красным.

Так как не все результаты по субсидии попадают в конкретный отбор, а только по выбранным КБК, то для организатора отбора есть возможность удаления результатов в рамках отбора и добавления ранее удаленного.  
По нажатию кнопку «+Результат» появляется поле с выпадающим списком ранее удаленных результатов, и кнопкой «Завершить выбор», как на <<id_image-g_os_2_21, Рисунке {counter:figure-number}>> и на <<id_image-g_os_2_22, Рисунке {counter:figure-number}>>.
