// настройка секции "Бюджет" в шаблоне заявок часть из описания настроек отбора

В секции {project_budget} в верхней части настраивается внешний вид информационного блока страницы {project_budget}. 

Поле «Максимально запрашивая сумма» – если {role_us} при заполнении бюджета превысит эту сумму — ему будет выходить ошибка/предупреждение (1).
Шкала «Нижний порог софинансирования бюджета %» – перемещается ползунок на показатель в процентах, цифра указывается справа (2), как на <<id_image-g_os_spb_1, Рисунке {counter:figure-number}>>.

[[id_image-g_os_spb_1]]
.Рисунок {counter:figure-number}. Внесение ограничений по отбору
image::g_os_spb_1.png[{image-size2}, align="center"] 

Следующие поля настраивают оформление виджета в начале секции, все вводится вручную, с возможностью использования стилей текста:

* «Заголовок секции»;
* «Заголовок общей суммы расходов»;
* «Заголовок общей суммы расходов по дополнительному полю 1, 2, 3»;
* «Заголовок общей суммы расходов по софинансированию»;
* «Заголовок общей суммы расходов по запрашиваемой сумме».

Изменения затрагивают выделенные области, как на <<id_image-g_os_spb_2, Рисунке {counter:figure-number}>>.

Изначально в настройках статей включено поле 1 с названием «Привлеченные средства, руб» и поле 2 с названием «Собственные средства, руб», а 3–е отключено, как на <<id_image-g_os_spb_3, Рисунке {counter:figure-number}>>.

При включенном переключателе, во время внесения пользователем сроков проекта в секции {project_plan} в секции {project_budget} ему динамически сформируется, то количество страниц для заполнения, сколько лет он указал. 
При выключенном переключателе для пользователя будет отображаться одна страница на весь период.

Рассмотрим настройку статьи. 
В верхней части статьи возможно изменить название, для этого необходимо нажать иконку image:{iconsdir}edit.png[width=16,height=16], как на <<id_image-g_os_spb_5, Рисунке {counter:figure-number}>>.

После нажатия кнопку «Добавить» появится новая статья расходов идентичная предыдущим, с возможностью ее настроить.

Для вызова меню настройки статьи необходимо нажать на иконку image:{iconsdir}more_vertical.png[width=16,height=16] справа от названия статьи. 
В открывающемся функционале можно изменить направленность статьи расходов, добавить подстатью и удалить статью с помощью соответствующих кнопок, как на <<id_image-g_os_spb_12, Рисунке {counter:figure-number}>>.

[[id_image-g_os_spb_12]]

Если необходимо добавить подстатью, то после нажатия соответствующей кнопки появляется модальное окно «Добавить подстатью», как на <<id_image-g_os_spb_12.1, Рисунке {counter:figure-number}>>.
При добавлении подстатьи заполняются следующие поля:

* «Объём собственных средств заявителя в процентах» — можно написать вручную или установить с помощью ползунка.
* «Заполнить общую сумму вручную» — при активации данного переключателя заявитель вручную заполняет поле Общая сумма.
* «Разрешить добавление расходов в статью» — при активации данного переключателя заявитель сможет добавлять расходы в статью. 
При деактивации этого переключателя, появится подсказка: «Если вы используете запрет на добавление строк в таблицу заявителями, не забудьте добавить строки расходов по умолчанию, иначе заявители не смогут подать заявку с незаполненной статьей.»
* «Название» — обязательное поле с названием подстатьи. 
* «Описание» — описание подстатьи расходов.
* «Стоимость единицы в рублях и количество» – стоимость одной единицы в рублях и множитель для подсчёта общей суммы.
* «Единица измерения» — обязательное для заполнения поле с единицами измерения. Можно выбрать несколько.
* «Комментарий и подсказка к статье» — текстовые поля, где можно оставить дополнительную информацию о подстатье, которая будет отражаться в заявке для пользователей.

Для настройки колонок статьи необходимо навести курсор мыши на иконку image:{iconsdir}more_vertical.png[width=16,height=16] статьи (1) и в меню выбрать иконку image:{iconsdir}edit.png[width=16,height=16] (2), как на <<id_image-g_os_spb_13, Рисунке {counter:figure-number}>>.

В открывшемся модальном окне производятся настройки строк статьи, где необходимо ввести название подстатьи и граф и добавить единицы измерения, как на <<id_image-g_os_spb_14, Рисунке {counter:figure-number}>>:

. «Объем собственных средств заявителя %» (1), с помощью ползунка, или вручную можно указать объем;
. «Название» (2), вручную ввести название подстатьи;
. «Единицы измерения» (3), поле с множественным выбором со справочником возможных единиц измерения;
. «Софинансирование (за весь период в рублях)» (4) содержит в настройках переключатель, при деактивации которого данное поле не будет отображаться;
. Кнопка «+Разделы» (5), отвечает за добавление дополнительных полей 1,2,3 «общей суммы расходов»;
. «Комментарий» (6) содержит в настройках переключатель, при деактивации которого данное поле не будет отображаться;
. «Подсказка к статье» (7) заполняется текстом подсказки по статье, она будет отображаться у участника при наведении курсора мыши на статью.

* Описание;
* Общая сумма (в рублях);
* Софинансирование (руб) (это поле тоже можно отключить в настройках);
* Запрашиваемая сумма.
