Подраздел {application_template} состоит из таблицы шаблонов. 
Таблица шаблонов состоит из следующих колонок:

* «ID» – уникальный номер шаблона задаётся автоматически при создании нового.
* «Дата создания» –дата создания шаблона.
* «Название» – шаблон заявок для подачи в электронном виде имеет название «Новый шаблон» (название является ссылкой для перехода в настройки шаблона).
* «Тип заявителя» – у шаблонов заявок для подачи в электронном виде указан тип субъекта экономической деятельности, для которого предназначен этот шаблон (в таблице отображены шаблоны только для типов субъекта экономической деятельности, которые изначально указаны 
ifdef::role2[в субсидии).]
ifdef::role2_cz[в услугах).]
* «Статус» – статус шаблона имеет два варианта «активен» и «не активен», включается в настройках шаблона.
Активный шаблон — это тот шаблон, который будет доступен для заполнения при наступлении даты начала подачи заявки.

Шаблон, предназначенный для заполнения и подачи заявки заявителем в электронном виде на Портале, может быть трех подвидов в зависимости от типа субъекта экономической деятельности, это указано в колонке «Тип заявителя»:

* «Юридическое лицо» – шаблон настроен с учётом требований по полноте информации, предъявляемых к юридическим лицам.
* «Индивидуальный предприниматель»– шаблон настроен с учётом требований по полноте информации, предъявляемых к индивидуальным предпринимателям.
* «Частное лицо» – шаблон настроен с учётом требований по полноте информации, предъявляемых к частным лицам (физические лица – производители товаров, работ, услуг).

При активном отборе и приёме заявок, во время авторизации, участник отбора выбирает из своей учётной записи пользователя на Едином портале государственных и муниципальных услуг роль и (или) организацию от имени которой предполагает работать на Портале. 
В зависимости от его выбора (ИП, ЮЛ или ФЛ) при создании заявки ему открывается активный шаблон, соответствующий его роли.

Возможно изменение настроек шаблона без изменения его основной части. 
Для этого необходимо нажать на название-ссылку, что откроет окно «Настройки шаблона заявки», как на <<id_image-g_os_1_3_2_2, Рисунке {counter:figure-number}>>.
